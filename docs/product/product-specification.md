## 1. System Functionalities
  
  ## 1.1 Authentication

Login
  =>Users can log into the system using their registered email and password.

Login fields:

*Email
*Password
*Role (Student / Teacher)
*Login Button

  =>After login, users are redirected to their respective dashboard.

Sign Up
   =>Users can create a new account.

student Sign-up includes:

*Full Name
*Email
*Password
*Confirm Password
*Student ID
*Department
*Phone Number

Teacher sign-up includes:

*Full Name
*Email
*Password
*Confirm Password
*Phone Number

 =>The system validates the information before creating the account.

  ## 1.2 Student Functionalities

Student Dashboard
   =>After logging in, students access the dashboard which contains:

*View QR Code
*Logout
*Generate QR Code

  =>Each student has a unique QR code generated by the system.

  =>The QR code contains the student's unique identifier. Teachers scan this code to record attendance.

  ## 1.3 Teacher Functionalities

Teacher Dashboard
  =>After login, teachers access a dashboard that displays:

*List of assigned courses
*Attendance management options
*View attendance records

Select Course
  =>Teachers must select a course before starting attendance.

The course list shows:
*Course Name
*Year
*Semester
*Start Attendance Session

  =>Teachers start an attendance session for a selected course. Once the session starts, the QR scanner is activated.

Scan Student QR Codes
  =>The teacher scans student QR codes using the mobile camera.

When a QR code is scanned:

*The system reads the student ID.
*The system verifies the student belongs to the course.
*The system records attendance for that student.
*Attendance Confirmation

  =>After scanning a QR code, the system displays a confirmation message showing:

*Student Name
*Student ID
*Attendance Status
*View Attendance Records

Teachers can review attendance records for any course.

Records include:

*Student Name
*Student ID
*Attendance Status
*Date

Teachers can filter attendance records by course or date.
